🧠 Architecture

Built using a strict MVC structure:

Controllers → request handling & routing
Services → business logic (cart, wishlist, pricing)
DAOs → database interaction (prepared statements)
Entities → domain models
Views → HTML with minimal PHP

✔️ Clear separation of concerns
✔️ No HTML in backend logic
✔️ Secure database interaction

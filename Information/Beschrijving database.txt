	Inhoud
De database is gevuld met data zodat elke groep aan de slag kan.
Trigger
Een trigger is een programma met SQL instructies. De trigger wordt gekoppeld aan een insert, update of delete instructie en een table. De trigger start als die actie gebeurt met de table.
bv. elke artikel heeft een uniek ean nummer. 
De waarde van het ean nummer wordt gegenereerd tijdens de toevoeging (insert) van een nieuw artikel in de table artikelen.
Tables en rechten
De mogelijke rechten zijn Select, Insert, Update en Delete.

	Events
Een event is een programma met SQL instructies. Een event wordt uitgevoerd volgens een opgegeven tijdschema (éénmalig op een bepaald tijdstip / dagelijks / wekelijks / ... ).
Er zijn drie events met elk hun eigen taak.
* Elke dag om 8 uur:
De vervallen actiecodes worden verwijderd.
* Elke dag om 9 uur:
Artikelen waarvan de voorraad onder de minimumaantal komt automatisch aangevuld. De toegevoegde artikelen worden automatisch geplaatst in de table magazijnplaatsen.
* Elke dag om 10 uur:
De uitgaandeBestellingsStatusId van de vorige dag wordt verplaatst van Onderweg naar Geleverd.
